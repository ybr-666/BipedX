## 🛠️ 机械组 (Mechanical)

**核心逻辑**：外骨骼不是把铁绑在腿上。重点是**减重、拟人化关节映射、穿戴舒适度**。

| 周次 |                             主修                             |                             辅修                             |
| :--: | :----------------------------------------------------------: | :----------------------------------------------------------: |
|  一  | 熟悉3DSource、大国工匠等SW插件与齿轮配合、机械配合、干涉检查工具等SW内置功能的使用；入门Rhino（曲面绘制软件）；导入人体扫描模型（Dummy Model），学习如何围绕“人腿”进行建模。 | 学习Bambu Studio的使用；安装SW，学会基本的测量功能和改图方法，以及导出不同格式文件（SLDPRT/SLDPRW/PDF/DXF...）； 了解**人体工学**基础（大腿/小腿长度比例、关节活动范围ROM）#推荐资源：[GrabCAD人体模型库](https://grabcad.com/) [B站：人体工学设计基础](https://www.bilibili.com/) |

|  三  | 了解不同配合方式、加工方式与精度与其对应成本；学会画出符合行业标准（国标GB）的用于发给有些厂商加工的工程图；**仿真与有限元分析 (FEA)**： 1. 学习SW Simulation，对关键承重部件（如髋关节连接件）进行应力分析。 2. **减重优化**：在保证强度前提下，通过打孔或拓扑优化减轻重量（外骨骼对重量极敏感）。**产出**：对第二周设计的伸缩腿进行静力学分析，输出一份“应力云图”，证明站立时不会断裂。 | 了解国标（GB），预习《大学物理》中力、热、电部分；了解电路板安装孔位标准（M2/M3螺丝孔距）。 预习：外骨骼的**力传导**路径。#推荐资源：[SW Simulation入门](https://www.google.com/search?q=https://www.bilibili.com/video/BV1MK4y1p7H7) |
