# 下肢关节活动范围 (ROM) 数据表

**说明**：下表数据为健康成年人在标准测量体位下的典型关节活动角度范围。所有数值单位均为度（°），0°定义为解剖中立位。

## 髋关节

| 运动方向 | 活动范围  | 备注                                         |
| -------- | --------- | -------------------------------------------- |
| **屈曲** | 0° – 120° | 膝伸展时约为0°-90°；膝屈曲时可达120°或更大。 |
| **伸展** | 0° – 20°  | 即过伸。部分文献记录为0°-15°或0°-30°。       |
| **外展** | 0° – 45°  | 骨盆固定下的侧向移动。                       |
| **内收** | 0° – 30°  | 腿向身体中线移动。                           |
| **内旋** | 0° – 35°  | 髋屈曲或伸展位测量值略有不同，此为常用参考。 |
| **外旋** | 0° – 45°  | 通常外旋范围大于内旋。                       |

## 膝关节

| 运动方向 | 活动范围         | 备注                                                        |
| -------- | ---------------- | ----------------------------------------------------------- |
| **屈曲** | 0° – 135° (150°) | 最大被动屈曲范围可达150°或以上。                            |
| **伸展** | 0°               | 完全伸直为0°，部分个体存在过伸（膝反张，记为负值，如-5°）。 |

## 踝关节

| 运动方向 | 活动范围 | 备注                                     |
| -------- | -------- | ---------------------------------------- |
| **背屈** | 0° – 20° | 膝关节伸直时，因腓肠肌紧张，角度会减小。 |
| **跖屈** | 0° – 50° |                                          |
| **内翻** | 0° – 35° | 距下关节主导，复合内收与跖屈。           |
| **外翻** | 0° – 15° | 距下关节主导，复合外展与背屈。           |

## 足部关节

| 关节               | 运动方向  | 活动范围            | 备注                               |
| ------------------ | --------- | ------------------- | ---------------------------------- |
| **跖趾关节 (MTP)** | 背伸      | 0° – 70° (第一趾)   | 第一跖趾关节对步态推进期至关重要。 |
|                    |           | 0° – 40° (其余四趾) |                                    |
|                    | 跖屈      | 0° – 45°            |                                    |
| **趾间关节 (IP)**  | 屈曲/伸展 | 约 40° – 60°        | 近端与远端趾间关节活动度总和。     |
